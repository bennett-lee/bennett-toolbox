# 行为准则

Bennett Toolbox 希望保持实用、友好和专注的协作氛围。本文适用于 Issue、
拉取请求和项目讨论。

## 期望行为

参与者应做到：

- 尊重他人，保持建设性沟通。
- 针对问题和方案讨论，而不是针对个人。
- 提供足够上下文，让别人能理解问题。
- 接受维护者关闭偏离主题或信息不足的问题。

## 不接受的行为

以下行为不被接受：

- 骚扰、威胁或人身攻击。
- 歧视性语言或图像。
- 未经允许公开他人隐私信息。
- 反复干扰技术讨论。

## 处理方式

维护者可以删除评论、关闭 Issue、限制用户参与，或采取其他合理措施来维护
项目秩序。

如需报告行为准则相关问题，请发送邮件至 bennettmail@163.com，并附上相关
链接和上下文。
