# 支持说明

本文说明如何获取 Bennett Toolbox 的使用支持，以及反馈问题时需要提供哪些
信息。

## 获取帮助

可复现的缺陷、功能建议和打包问题请通过 GitHub Issues 提交：

```text
https://github.com/bennett-lee/bennett-toolbox/issues
```

提交前，请先确认你使用的是最新安装包，并检查问题是否仍然存在。

## 建议提供的信息

反馈问题时请尽量提供：

- 操作系统和版本。
- 应用版本或提交 SHA。
- 涉及转换功能时的输入文件类型。
- 错误信息或截图。
- 可以复现问题的具体步骤。
