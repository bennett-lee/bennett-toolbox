# 更新日志

Bennett Toolbox 的重要变化会记录在此文件中。

项目早期阶段暂时使用简单的版本分组记录，后续发布稳定版本时再补充更严格
的版本规则。

## 1.0.1

此版本规范了发布版本号，并补充近期新增能力。

- 图片转换支持批量选择、批量转换、转换进度显示和 ZIP 打包下载。
- 新增 PDF 转图片功能，可将每页 PDF 转为 PNG 或 JPG。
- 新增图片去水印功能。
- 新增视频转 GIF 功能。
- 新增 Windows HEIC/HEIF 转换能力，使用随应用安装的 `heic-convert` 运行时。
- 新增 GitHub Actions 自动构建和发布 macOS、Windows 安装包。
- 新增文档转 Markdown 转换进度条。
- 新增内置 MarkItDown 支持，用户不再需要全局安装 MarkItDown。
- 补齐 GitHub 项目规范文件、Issue 模板、拉取请求模板和 Dependabot
  配置。

## 1.0.0

这是 Bennett Toolbox 的第一个打包版本。

- 新增 JSON 格式化、变量命名、图片转换、文件搜索、文档转 Markdown、截图
  悬浮和屏幕取色功能。
